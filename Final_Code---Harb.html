lyrics$lyrics <- gsub( " ", " ", lyrics$lyrics )

Now I am going to use lapply to count the number of words in the songs. Im only going to count the first six, only because I want to show the function, and Id rather not have a whole sheet of numbers.

first.six.songs <- head( lyrics$lyrics )

Now we are going to build a word cloud! For this Im going to look at the song titles, as they will be more telling I think of the nature of the songs. First we are going to use Corpus() from the text mining package. I found this website to be very instructional: http://www.sthda.com/english/wiki/text-mining-and-word-cloud-fundamentals-in-r-5-simple-steps-you-should-know#:~:text=The%20procedure%20of%20creating%20word,keywords%20as%20a%20word%20cloud.

docs <- Corpus(VectorSource(song_name_word))


dtm <- TermDocumentMatrix(docs)
m <- as.matrix(dtm)
v <- sort(rowSums(m),decreasing=TRUE)
d <- data.frame(word = names(v),freq=v)
head(d, 10)


Now that all of the background stuff is done, we are goinf to make the cloud! Woohoo!

set.seed(12345)
wordcloud(words = d$word, freq = d$freq, min.freq = 20,
          max.words=200, random.order=FALSE, rot.per=0.6, 
          colors=brewer.pal(12, "Paired"))
